# Mutualy 📈

A mutual fund search and tracking web app built as a company internship assignment.  
Users can search mutual fund schemes, view NAV details, and save funds to their personal profile.

## 🚀 Features
- Search mutual fund schemes by company/name
- View detailed NAV and fund information
- Save funds to personal watchlist
- User authentication (Register/Login)
- CRUD operations on saved funds
- Search filtering and sorting

## 🛠️ Tech Stack
**Frontend:** React.js, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose  
**Auth:** JWT  
**API:** [MFAPI](https://api.mfapi.in/mf) — Free Indian Mutual Funds API

## 📌 Context
> Built as an internship selection assignment — implemented full auth flow, external API integration, and CRUD with search/filter functionality.
